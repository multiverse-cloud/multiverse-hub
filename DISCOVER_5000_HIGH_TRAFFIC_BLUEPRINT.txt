======================================================================
1. IMPORTANT REALITY
======================================================================

Do NOT chase only impossible head terms like:
- top movies
- best songs
- top actors

Those are huge competition terms.

The winning strategy is:
- high-volume long-tail
- language + genre + use-case combinations
- strong internal linking
- lots of useful editorial pages

So the target is not:
- 5000 random pages

It is:
- 5000 structured search-intent-driven content records


======================================================================
2. WHAT COUNTS AS A "DATA UNIT"
======================================================================

For this project, 5000 data units can include:
- ranking pages
- guide pages
- collection pages
- subcategory hubs
- editorial support pages
- FAQ groups
- relation units between pages

Recommended practical split:

Tier A
- 600 ranking pages

Tier B
- 500 guide pages

Tier C
- 700 category / subcategory / hub pages

Tier D
- 2200 curated ranking items across all pages

Tier E
- 1000 FAQ / related / support data units

Total:
- 5000+ structured content units

This lets the admin system grow realistically instead of trying to hand-write 5000 standalone full pages immediately.

3. SEARCH DEMAND STRATEGY
======================================================================

Focus on these 4 query layers.

Layer 1: High-traffic broad but still achievable
- best tamil movies
- best korean dramas
- top thriller movies
- best family movies
- best romantic songs

Layer 2: High-volume long-tail
- best tamil thriller movies for weekend
- best korean dramas for beginners
- top road trip songs tamil
- best comedy series to binge watch
- best family movies to watch with parents

Layer 3: Intent-rich helper queries
- what to watch this weekend
- how to start korean dramas
- how to choose a good thriller movie
- best songs for long drive
- best directors for beginners

Layer 4: Entity cluster queries
- best movies of [director style]
- best performances by [actor type]
- best actress movies for family watch
- best crime series like [theme]

10. INTERNAL LINKING PLAN
======================================================================

Every page should link to:
- 3 to 6 related rankings
- 1 to 3 guides
- 1 parent hub page

Example:

Page:
- best tamil thriller movies

Should link to:
- best tamil family movies
- best crime series worth bingeing
- best directors for thriller lovers
- how to pick a weekend thriller
- tamil movies hub

This is what builds topical authority.


======================================================================
11. SEO RULES FOR HIGH-TRAFFIC TARGETING
======================================================================

Do:
- use clear H1
- keep slug short
- put exact search intent in title
- keep meta description readable
- add methodology
- add FAQs matching real user intent
- keep page useful and scannable

Do not:
- stuff keywords repeatedly
- create near-duplicate pages with only one word changed
- auto-generate fake content
- publish pages with only a title and 5 names


======================================================================
12. EDITORIAL WRITING RULES
======================================================================

Each ranking item summary should answer:
- what kind of audience it fits
- what vibe it has
- why it is on this list

Each guide step should answer:
- what the user should do next
- how to choose between options
- what to avoid

Each FAQ should answer real search intent:
- is this beginner friendly
- is this family safe
- what if I want shorter options
- what if I prefer thrillers over romance
