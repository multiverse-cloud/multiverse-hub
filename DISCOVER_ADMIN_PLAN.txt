10. ADMIN API CONTRACT
======================================================================

10.1 GET /api/admin/discover

Returns:
- { lists: DiscoverList[] }

10.2 POST /api/admin/discover

Case A: seed starter data
Body:
- { "action": "seed" }

Returns:
- { "success": true, "count": number }

Case B: create/save new list
Body:
- { "list": DiscoverList }

Returns:
- { "success": true }

10.3 PATCH /api/admin/discover

Body:
- { "list": DiscoverList }

Returns:
- { "success": true }

10.4 Error expectations

Unauthorized:
- status 401
- { "error": "Unauthorized" }

Firebase missing for writes:
- status 500
- { "error": "Firebase Admin is not configured for discover persistence." }

Missing payload:
- status 400
- { "error": "Missing discover list payload" }
